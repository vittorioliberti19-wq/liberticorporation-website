# Design System Guidelines

Welcome to the foundational design system. These guidelines describe the core aesthetic and functional principles for all user interfaces and experiences.

## Theme Overview

This design system is built with a **dark color mode**, emphasizing visual comfort and reduced eye strain in low-light environments. The overall aesthetic is clean and modern, leveraging a consistent typography system and balanced visual elements.

## Color Palette

The color palette is meticulously crafted to ensure accessibility, hierarchy, and brand recognition.

-   **Primary Color**: `#1275e2` - A vibrant blue, serving as the brand's most distinctive chromatic color. Ideal for primary calls to action, active states, and key interactive elements.
-   **Secondary Color**: `#5f78a3` - A supporting grayish-blue, used for less prominent UI elements, chips, and secondary actions, providing a harmonious contrast to the primary color.
-   **Tertiary Color**: `#c55b00` - A warm orange accent color, used for highlights, badges, notifications, or decorative elements, adding a touch of vibrancy without overpowering the primary palette.
-   **Neutral Color**: `#74777f` - A foundational gray, providing a stable base for backgrounds, surfaces, text, and non-chromatic UI elements, ensuring readability and visual balance.

## Typography

A consistent and legible typography system is crucial for effective communication. The system utilizes a single font family across various roles to maintain visual unity.

-   **Headline Font**: `Inter` - Used for all headlines and prominent textual elements, ensuring strong visual impact and readability.
-   **Body Font**: `Inter` - Applied to all body text, paragraphs, and general content, providing excellent legibility for extended reading.
-   **Label Font**: `Inter` - Dedicated to labels, small text elements, and annotations, maintaining consistency with the overall typographic rhythm.

## Roundedness

The corners of UI shapes, such as buttons, cards, and input fields, feature a **moderate roundedness (2)**. This strikes a balance between sharp, modern lines and a softer, more approachable aesthetic, contributing to a friendly and contemporary feel.

## Spacing

The design system incorporates a **normal spacing (2)** approach. This ensures a balanced level of whitespace both within and between UI elements, leading to clear separation, improved readability, and an organized visual layout. It provides sufficient breathing room without making the interface feel overly sparse or dense.
